# Company Management System

Academic project developed in the first year of university, implemented in the C programming language.

This is a console-based application that allows the management of company-related information through an interactive, menu-driven interface, with a strong focus on modular programming and structured code design.

---

## Project Objectives

This project was developed with the objective of consolidating fundamental programming concepts in C, namely:

- Structured and procedural programming
- Modularization using header and source files
- Separation of concerns within the codebase
- User interaction via text-based menus
- Basic data handling and validation techniques

---

## Main Features

- Interactive menu system in the console
- Distinct functionalities for users and administrators
- Management of company entities through structured logic
- Modular architecture for improved readability and maintenance
- Basic validation of user input

---

## Project Structure

main.c                  – Program entry point and initialization  
menus.c / menus.h       – Menu logic and navigation system  
management.c / .h       – Core management functionalities  
administrator.c / .h   – Administrator-related operations  
user.c / user.h         – User-related operations  
Makefile                – Compilation automation  
build/                  – Build artifacts generated by the IDE  
nbproject/              – NetBeans project configuration files  

---

## Compilation and Execution

### Requirements

- C compiler 
- Make utility
- Compatible environment (Linux, macOS, Windows with MinGW, Cygwin or WSL)

### Compile the project

From the project root directory, run:

```bash
make
```

### Run the application

After compilation, execute:

```bash
./main
```
